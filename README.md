# 🍕 Pizza Calc Pro — Da Marcolino

Una calcolatrice premium per ordini di pizzeria, progettata con un'estetica moderna ispirata alle **tavole periodiche** e alle **calcolatrici meccaniche**.

![Pizza Calc Pro Snapshot](index.html) <!-- Note: This is a placeholder for a screenshot if available -->

## ✨ Caratteristiche principali

- **Interfaccia a Tabella Periodica**: Le pizze sono organizzate come elementi chimici, con colori distintivi per categoria (**Classiche**, **Calzoni**, **Speciali**).
- **Design Premium**: Tema scuro con display LCD stile CRT (fosfori verdi) e pulsanti con micro-animazioni meccaniche.
- **Gestione Ordini Avanzata**:
  - Aggiunta rapida di supplementi (Doppio Impasto con incrementi di +€1.00 e +50c).
  - Opzioni di trasporto personalizzabili (€1 / €2).
  - Cronologia ordini visibile in tempo reale sul display.
- **Responsive**: Ottimizzato per l'uso su Tablet e Smartphone in ambiente di lavoro frenetico.
- **Persistenza**: Salva automaticamente lo stato del trasporto e della comanda nel browser.

## 🚀 Tecnologie utilizzate

- **HTML5 & CSS3**: Layout moderno con CSS Grid/Flexbox e variabili per il design system.
- **Vanilla JavaScript**: Logica reattiva senza dipendenze esterne.
- **SVG**: Logo vettoriale personalizzato integrato.

## 📖 Come usarlo

1. Apri `index.html` in qualsiasi browser moderno.
2. Naviga tra le categorie usando i tab in alto.
3. Clicca su una pizza per selezionarla.
4. Aggiungi eventuali extra dalla sezione in basso.
5. Usa i controlli rapidi (icone 🍞 e 🚚) per supplementi e trasporto.
6. Clicca **INVIO** per aggiungere alla comanda o **CANC** per annullare la selezione corrente.
